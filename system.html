HOW DOES THE SYSTEM WORK

  HIGHLIGHTS

  - Very light-weight agent, less than 10 MB RAM
     and 2% CPU utilization.

  - All secured communication between
     agent/administrator & server side.

  - Agent Auto-Start after installation.

  - Supports Multi-Session for windows Servers.

  - All Administration tasks done through the web
     interface (GUI).

  - Requires a Help-Desk password when
     uninstallation.

  - Process-Kill prevention.

  - Granular Policy Control.

  - Ability to upload Users & Groups from your
     existing Active Directory environment.
